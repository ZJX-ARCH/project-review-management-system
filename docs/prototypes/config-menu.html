系统配置管理

配置说明：
系统支持灵活配置不同项目类型的流程和评审标准，满足多样化的项目管理需求。管理员可以根据实际业务需要，自定义项目类型、评审模板、审批流程等关键配置。
📋
项目类型管理
配置不同的项目类型及其属性，如产品研发、技术创新、流程优化等。每种类型可设置不同的申报要求、预算范围、审批流程等。
核心配置
⭐
评审模板管理
为不同项目类型配置专属的评审标准和评分项。可自定义评分维度、权重、满分值，确保评审的科学性和针对性。
核心配置
🔄
流程配置
配置项目从申报到立项的完整流程。可为不同项目类型设置不同的审批节点、审批人、流转规则等。
核心配置
🏢
部门管理
配置组织架构和部门信息，管理部门层级关系、人员归属、部门负责人等基础数据。
👨‍🏫
专家库管理
管理评审专家基础信息、专业领域、职称、评审权限等。支持专家分组和智能推荐功能。
🔐
权限管理
配置系统角色和权限，定义不同角色可访问的功能模块和数据范围，确保系统安全。
📧
通知模板
配置系统通知模板，包括邮件、短信、站内消息等。支持变量替换和多场景模板管理。
⚙️
系统参数
配置系统全局参数和业务规则，如文件上传限制、会话超时、密码策略等基础设置。
📚
数据字典
管理系统数据字典，维护各类枚举值和码表数据，如项目状态、审核结果、优先级等。
⚠️ 重要提示：
• 配置修改会影响系统业务流程，请谨慎操作
• 建议在系统维护时段进行重要配置的修改
• 所有配置操作都会记录操作日志，便于审计和追溯
• 部分核心配置修改后可能需要重启应用才能生效
